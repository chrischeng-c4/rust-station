# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| `Cmd+O` | Open project folder |
| `Cmd+W` | Close current project |
| `Cmd+Q` | Quit Rustation |

## Project Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+1` - `Cmd+9` | Switch to project 1-9 |
| `Cmd+[` | Previous project |
| `Cmd+]` | Next project |

## Worktree Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+[` | Previous worktree |
| `Cmd+Shift+]` | Next worktree |

## Tab Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+1` | Tasks tab |
| `Cmd+2` | Docker tab |
| `Cmd+3` | Settings tab |

## Actions

| Shortcut | Action |
|----------|--------|
| `Cmd+R` | Refresh current tab |
| `Cmd+K` | Clear output panel |
| `Cmd+C` | Copy selection |

## Development

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+I` | Open DevTools |
| `Cmd+Shift+R` | Force reload |
