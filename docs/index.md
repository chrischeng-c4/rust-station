## Quick Overview

Rustation is a native macOS desktop application built with Electron, React, and Rust. It provides a unified interface for common development tasks:

- **Projects**: Open multiple git repositories as tabs
- **Worktrees**: Create and switch between git worktrees
- **Docker**: Manage containers for local development
- **Tasks**: Run justfile commands with output streaming
