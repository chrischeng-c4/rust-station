# Features Overview

Rustation provides three main features for developer productivity.

## Project Management

Open and manage multiple git repositories simultaneously:

- **Tab-based navigation** - Each project gets a tab
- **Git worktree support** - Work on multiple branches at once
- **Recent projects** - Quick access to previously opened projects
- **Auto-detection** - Finds git root automatically

## Docker Dashboard

Visual interface for development containers:

- **Service status** - See running/stopped state at a glance
- **One-click controls** - Start, stop, restart services
- **Log viewer** - Real-time container logs
- **Database creation** - Create DBs in PostgreSQL/MySQL/MongoDB

## Task Runner

Execute justfile commands with real-time output:

- **Command list** - All available tasks from justfile
- **Output streaming** - Watch command output live
- **Status indicators** - See success/failure state
- **Per-worktree** - Each worktree has separate output
